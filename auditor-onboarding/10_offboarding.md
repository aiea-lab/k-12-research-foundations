    - (Instructions for CMPM118/Auditor program without an Independent Study) The final report should be a **technical document** that describes your project/experiments in a narrative (and cohesive) way. Please, focus only on the technical part, you can skip/ignore feedback, hugo, nautilus, and similar. Conversely, it should include the goal of the project, the algorithms you used, where and how you applied them, what you observed, results analyses and similar (not only these points!). The more the better. You should already have most of this information in your past deliverables, and you can absolutely use those as a base. However, the final report must be a standalone document that can be read and fully understood without having access to any other materials. There is no limit on the number of pages for this report because it should include plots and images but please include only technical details and don’t be oververbose (as an indication the text without any images shouldn’t be longer than 4-5 pages).
